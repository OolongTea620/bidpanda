# User 도메인

## 정의

회원의 정보를 저장, 조회, 수정 하는 기능

## 구현 목록

```text
- 회원 가입
- 회원 정보 조회
- 회원 정보 수정 (닉네임)
- 회원 탈퇴 (비활성화)
```

### 기능 명세

### 1. 회원 가입

**데이터**

```text
- email : 사용자 이메일, 이메일 형식을 만족해야 한다. 필수 입력, 중복 비허용
- password : 인증 인가를 위한 비밀번호. 필수 입력
- nickname : 회원 활동을 위한 가명,필수 입력, 중복 허용 
```

**기능**

```text
- 회원은 같은 이메일로 중복 가입이 불가능하다
- 비밀번호는 해시로 암호화 하여 데이터베이스에 저장한다.
```
